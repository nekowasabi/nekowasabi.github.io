20180910艦これ日報

暮らし

  • Keep
    • 漢方外来に行って相談。自分が飲んでいた漢方とはまったく違ったものを処方していただく。
      • そして保険がきくため尋常ではないと思ってしまうくらい安い。すばらしい。
      • 漢方の効き目はまだわからず。
        • 朝は痰を出そうという方針の漢方なため、起き抜けは症状が悪化しているように観察される。
          • が、それ以外は咳の量は減っているように感じる。
      • いずれにせよまだ数日しか漢方を飲んでいないため、まだまだ継続観察。
  • Problem
    • チンニングスタンドにモノが転がっているためトレーニングができない。
    • そもそもトレーニングそのものができていない。
      • TODOリストにあるものを実行するよう自分を訓練してきたため、リストにないものは存在しないため、やらないという理路だろう。
  • Try
    • 最優先でトレーニング用のTODOリストを作成する

艦これ

  • Keep
    • イベント開始。甲難易度E1で挫折。
  • Problem
    • 対潜装備が貧弱でボスへのダメージが低い。
      • 100以下まで削ることができなかったため、単純に地力不足。
    • 敗因
      • 三式装備がほとんどない。
        • 1-5、7-1ではなんとかなってしまったため、問題の露見が遅れた。
        • アドバイスを軽視して、先制対潜できる駆逐艦の育成を後回しにした。
          • 3-5の撃破率を重視して重巡の育成を優先したためで、善し悪しでもある。
      • 対潜支援が弱すぎる
        • 名前付き艦爆が最低ラインだったため、威力が出せなかった。
  • Try
    • 朝潮など先制対潜ができる駆逐艦を優先的に演習に出す
    • 海防艦はイベントの最初で使うケースが多いため、最低限改にはする。
    • 余裕があれば蒼龍牧場で名前付き航空機を集める
      • これは数ヶ月かかるので、そこまで優先度は高くない。できない。
    • イベント難易度は丙丁で固定
      • 最初の目的である『イベント完走』から動かさないようにする。
        • 勝ち目のある勝負だけをする、嫌になって投げ出す状況にならないことを最優先にする。
          • スパ帝氏の『勝負事で肝心なのは負けても死なないこと』というキーワードを思い出す。
        • 正直なところ「これだけ時間を使っても勝てないなら、艦これもう辞めるか」というキーワードが頭をよぎったので、条件がそろうと簡単に心が折れるだろうことがわかった。
          • 楽しくないゲームは積極的に手放したいものでしかないのだ。
